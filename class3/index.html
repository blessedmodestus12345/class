C I HUB SCHEME

Lesson Plan

Teacher:Viceo

Duration:First Month

OVERVIEW & PURPOSE

The first month would be used to teach student to get familiar with bsic web development environment and also basic HTML and CSS(Inline CSS for the first month).

WEEKLY SCHEDULE

Target Project: http://dribble.com/shots/14290034-Earth-Tribe

Week 1:

  1. How does the web work?(Explain web1, web2,web3.)
  2. Get an overview of what the learning curve would look like.
  4. Why do we use the technologies we us today? Setting up your local environment(Hack and Tips)
  5. HTML History and Basics

Week 2

1 Creating your first web page(HELLO WORLD).

2 HTML Elements.

  1. attribute.
  2. Paragraphs, Headings and comment.
  3. HTML formating Element.
  4. Quatation and Citation Element.
  5. Links, Images and Favicon.

3 HTML Contd

  1. Tables.
  2. HTML list
  3. Introducing Block level and Inline level Elements.
  4. Iframes.
  5. Computer Codes and Entities.
  6. Symbols and Emojis.

4 HTML charset, Url Encod and Abstract Knowledge(XHTML, XML).

Week 3

  1. HTML Forms
  2. HTML Mode.

Week 4

  1. Classes and IDs.
  2. Styling and start Targe Project.

OBJECTIVES

1 Learn the basics of web and get comfortable with seeing coding.
2 Students would get hands-on experience creating their own web page.

MATERIALS NEEDED

1 Projector(Once per week).

ACTIVITIES

1 One project per .
2 Become the teacher (30 mins session where the student will get to teach what he or she understands from the previous classes).
Simple Game that would make the student learn faster and would make so much fun.
